# About GS Error Reporter

GS Error Reporter is a lightweight email-first error monitoring solution for Next.js applications.

It automatically captures:

- Client-side runtime errors
- Server-side runtime errors
- Unhandled Promise Rejections
- Console Errors
- Console Warnings

and sends beautifully structured HTML email reports directly to the developer's inbox using Gmail SMTP.

The package currently supports:

- Next.js App Router
- Next.js Pages Router
- TypeScript
- JavaScript

The setup takes only a few minutes using the built-in CLI.

The philosophy of the project is:

"No dashboards.
No third-party monitoring service.
No subscriptions.
Just install, configure, and receive production error reports."
